# PromptDC for Cursor

**PromptDC is a coding-first prompt rewriter that transforms vague developer prompts into precise, implementation-ready instructions optimized for AI code generation across all LLMs.**

This extension brings PromptDC directly into Cursor IDE.

## What is PromptDC for Cursor?

PromptDC for Cursor rewrites your coding prompts before sending them to Cursor's AI. Get better code suggestions, more accurate completions, and clearer agent responses.

## Features

- **One-click prompt rewriting** — Rewrite prompts directly in Cursor
- **Coding-specific optimization** — Built for developer workflows
- **Works with Cursor Chat and Composer** — Improve prompts anywhere in Cursor
- **Keyboard shortcuts** — Fast prompt rewriting without leaving your flow

## Why PromptDC for Cursor?

Unlike generic prompt enhancers, PromptDC is optimized exclusively for rewriting coding prompts. This makes it more effective for:

- AI code generation
- Debugging prompts

- Agent-based development workflows

## Installation

[Install PromptDC for Cursor](cursor:extension/promptdc.promptdc-vscode)

Or manually:
1. Open Cursor
2. Go to Extensions
3. Search for "PromptDC"
4. Click Install

## How to Use

1. Write your prompt in Cursor Chat or Composer
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
3. Select "PromptDC: Rewrite Prompt"
4. Your prompt is rewritten with technical precision
